# AI Gesture Recognition System

This project demonstrates a Computer Vision based Gesture Recognition System using Python.

It includes two AI powered applications:

## Air Drawing System
Draw in the air using hand gestures captured by a webcam.

Features:
• Real time hand tracking  
• Gesture based drawing  
• MediaPipe hand landmark detection  

Tech Stack:
Python, OpenCV, MediaPipe

---

## AI Emotion Persona Overlay

Detects facial emotions and overlays persona filters in real time.

Features:
• Emotion detection  
• Face tracking  
• Persona overlay filters  

Tech Stack:
Python, OpenCV, Deep Learning

---

## Installation

Install dependencies:

pip install -r requirements.txt

Run the Air Drawing project:

python air_drawing.py
